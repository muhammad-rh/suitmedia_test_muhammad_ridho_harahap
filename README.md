# Flutter Palindrome

## Link to download apk

https://bit.ly/palindrome-apk

## Feature

This application offers a Palindrome text checking feature and user selection sourced from regres.in

- google_fonts: ^3.0.1
- provider: ^6.0.3
- dio: ^4.0.6
- fluttertoast: ^8.0.9
